Google's API

Die API von Google ermöglicht es Entwicklern, auch wenn sie nicht für Google arbeiten, verschiedene Dienste von Google in ihre Produkte einzubauen.

So kann man z.B. auf einer Seite, die nicht von Google stammt, eine kleine Karte anzeigen lassen, die beim Anklicken direkt Google Maps öffnet.

Durch geschickte Anfragen an den Google-Server habe ich ein Programm geschrieben, das eine Excel-Datei mit den Unternehmen ausgibt, die noch keine Website bei Google Maps eingetragen haben.

Man kann dabei nach Branche und Ort filtern.

Dieses Programm könnte für Webdesigner interessant sein, um Kunden zu generieren.

Die Anfrage, die im Video zu sehen ist, geht alle Restaurants in der Umgebung von Bamberg durch und speichert diejenigen, die noch keine Website haben.
