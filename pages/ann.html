Multi-Layer Neural Network

Die Stanford University veröffentlichte einen Artikel, der alle wichtigen mathematischen Formeln zur Modellierung eines mehrschichtigen neuronalen Netzwerks (ANN) enthielt. Da ich zu dieser Zeit eine Vorlesung über maschinelles Lernen besuchte, nahm ich es mir zur Herausforderung, die mathematischen Formeln in funktionierenden Code umzusetzen.

Um ein KI-Modell schnell aufzusetzen, gibt es gute frei verfügbare APIs. Mit ein paar Zeilen Code definiert man, welches Modell auf welchen Daten trainiert werden soll. Aus anderen Projekten weiß ich, dass dieser Ansatz recht einfach umzusetzen ist.

Da ich hier aber die Fähigkeit trainieren wollte, abstrakte Formeln in Code umzuwandeln, habe ich keinen vorgefertigten Code verwendet.


Ergebnis

Das fertige ANN schnitt bei klassischen Benchmark-Datensätzen gut ab. Funktionen wie das Laden/Speichern von Gewichten, Training oder Testen sind voneinander getrennt und voll funktionsfähig.

Probleme, für die diese Art des maschinellen Lernens geeignet ist, umfassen Bilderkennung, Signalverarbeitung, Finanzanalyse und viele mehr.
